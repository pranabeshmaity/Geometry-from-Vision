# Geometry from Vision

## Overview

This project explores the fundamentals of 3D vision, focusing on mesh, point cloud, and voxel representations. It demonstrates how 2D image data can be transformed into meaningful 3D structures.

---

## Key Concepts

* **Mesh**: A structured representation using vertices, edges, and faces
* **Point Cloud**: A set of 3D points representing object geometry
* **Voxel Grid**: A volumetric representation using 3D pixels

---

## Features

* 3D mesh creation and rendering using PyTorch3D
* Point cloud generation and visualization
* Voxel-based reconstruction
* Basic 2D to 3D transformation pipeline

---

## Project Pipeline

1. Understand different 3D representations
2. Generate and visualize meshes
3. Convert data into point clouds
4. Build voxel-based structures
5. Explore simple 2D → 3D reconstruction
